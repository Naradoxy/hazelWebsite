Dead Mice (Unreal 2025)

Dead Mice was the capstone project my team did when I was in the Circuit Stream Game Design course offered through the University of Toronto. The game is an autoshooter roguelike set in a rave during a zombie mice apocalypse. I did all the in engine work and the more technical game design. We had four weeks to do everything.
